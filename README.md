# cloud-music-manager
nas(群晖)音乐管理器,配合**music-tag-web**,**navidrome**,**音流**使用

## Features

- [ ] 支持telegram交互
- [ ] 支持docker部署和Github Action
- [ ] 根据平台下载音乐解锁程序(自动)
- [ ] 集成OpenAi
- [ ] 整理
    - [ ] 音乐库重复性判断(自动)
    - [ ] 音乐下载,同时支持本地音乐上传
    - [ ] 音乐解锁(自动)
    - [ ] 支持手动修改音乐标签
    - [ ] 上传到nas(自动)
- [ ] 入库通知
    - [ ] 如入库失败 告知原因
    - [ ] 如入库成功 发送音乐名 歌手 专辑名
